# 双重效应原则

> 评价一项有害副作用的行动时，区分伤害是实现目的的手段，还是被预见但并非所求的副作用。

## 它解决什么问题

同样是“为了较大利益而造成伤害”，行动者是否把伤害当作达成利益的手段、是否符合比例、是否寻找更少伤害的替代，会影响道德判断；只比较人数会遗漏这些差异。

## 怎么起作用

```text
行动同时产生好结果与伤害 → 检查意图、手段、副作用、比例和替代方案 → 区分不同的行动结构 → 改变可允许性判断
```

## 参与角色

| 角色 | 作用 |
|---|---|
| 行动者 | 选择行动并形成目的与手段 |
| 受益者 | 获得行动的好结果 |
| 受害者 | 承担作为手段或副作用的伤害 |
| 审查者 | 检查比例、替代方案和伤害最小化 |

## 什么时候成立

- 同一行动可预见地同时产生重要收益和严重伤害。
- 能有意义地区分行动目的、实现手段与副作用。
- 已检查比例、较少伤害的替代方案和伤害最小化。

## 什么时候失效

- 仅说“我没有恶意”不能把可预见伤害自动变成可接受副作用。
- 意图与手段的边界可能有争议，原则本身也不能解决全部电车案例。
- 对权利、分配、历史责任与程序缺失，它没有完整答案。

## 新场景测试

### 合理迁移

医生为阻止感染而截肢，截肢伤害是治疗手段的一部分；与给药后可预见但非治疗所需的副作用，在行动结构上不同，仍需比例与替代审查。

### 错误迁移

管理者明知产品会伤害用户，却只因口头宣称“目标是增长”便把伤害称作副作用；若伤害实际上是盈利路径的一部分，声明不能改变结构。

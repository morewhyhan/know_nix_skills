# 输入：要不要打开分洪闸

暴雨来临前，某市模型预测：如果保持分洪闸关闭，五个低洼社区约五千人可能遭受中等程度的住宅和财产损失；如果打开分洪闸，多数城区可保住，但下游一个约六百人的村庄很可能遭受更深的积水，住宅、耕地和一处祖坟区可能受损。

模型对雨量、堤坝承压和两种方案的实际损失区间仍有明显不确定性。下游村庄过去已经多次承担城市工程的环境代价，这次也没有参与预案制定。市政府承诺事后补偿，但补偿标准、到账时间和不可货币化损失的处理都没有确定。

会议上有人说：“这就是电车难题。牺牲六百人，保护五千人，数字已经说明答案。”也有人说：“只要有人会因主动开闸受损，就绝不能开。”

请分析这段材料，把可复用机制写入知识库，并判断能否组合出一个比二选一更有用的决策系统。

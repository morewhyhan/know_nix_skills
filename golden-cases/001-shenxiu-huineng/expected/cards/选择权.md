# 选择权

> 有权尝试，但没有义务在结果不利时继续；结果有利时仍能保留上行收益。

## 它解决什么问题

面对无法准确预测的未来，与其押中唯一答案，不如保留多个低成本入口，并在获得信息后决定是否继续。

## 怎么起作用

```text
未来结果不确定
→ 用有限成本取得尝试权
→ 观察结果后选择继续、扩大、停止或放弃
→ 下行损失受初始成本限制，上行仍可保留
```

## 参与角色

| 角色 | 作用 |
|---|---|
| 决策者 | 取得继续、扩大、停止或放弃的权利 |
| 选项 | 用有限成本提供未来行动入口 |
| 不确定环境 | 在后续揭示哪条路径更有利 |

## 什么时候成立

- 放弃和停止是真实可执行的。
- 初始成本、持续成本和退出成本有上限。
- 成功后仍有能力扩大，而不是收益被他人锁定。
- 多个选项不会共同暴露于同一次毁灭性风险。

## 什么时候失效

- 名义上可退出，实际存在巨额沉没成本或声誉绑架。
- 所有选项高度相关，危机时一起失败。
- 频繁试验的累计成本超过偶发上行收益。

## 新场景测试

先发布三篇不同方向的短内容，再把真实反响最好的一个扩成长文，比一次性投入一个未经验证的大选题拥有更多选择权。

## 来源与证据

- 不是由两首偈直接提取。
- 作为反脆弱系统的组成机制引入。
- 待核验：与金融期权、实物期权和塔勒布所说 optionality 的精确边界。

## 关系

- `组成` → [[反脆弱]]：理由：帮助形成下行受限、上行开放的收益结构；边界：选择权存在不等于波动增加必然提高净收益；证据：独立材料支持与系统推导。
- `组成` → [[保护核心，开放试错]]：理由：试验边需要停止、回滚和扩大成功路径的权利；边界：不适用于不可逆核心；证据：组合设计。

## 候选关联

- 实物期权可能提供更精确的投资语言，需核验是否覆盖非金融使用。

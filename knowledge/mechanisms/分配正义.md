# 分配正义

> 不只看总收益，还看收益、成本和风险分别落在谁身上。

## 它解决什么问题

同一个“总体更优”的方案可能让收益分散给多数人，却把高强度或不可逆的伤害集中给少数人。只看总量会遮住这种差异。

## 怎么起作用

```text
政策改变成本与收益的去向 → 群体暴露和承受能力不同 → 同一总量产生不同个体后果 → 公平性、接受度和最终福利随分配改变
```

## 参与角色

| 角色 | 作用 |
|---|---|
| 决策者 | 决定怎样计算和分配风险 |
| 获益者 | 获得风险降低或资源收益 |
| 受损者 | 承担直接、集中或不可逆损失 |
| 执行与补偿机构 | 决定承诺能否实际兑现 |

## 什么时候成立

- 方案会把成本、收益或风险分配给可区分的群体。
- 群体的暴露程度、脆弱性或恢复能力不同。
- 决策者能够识别至少一部分分配结果。

## 什么时候失效

- 不能仅凭群体身份预设每个人的真实偏好。
- “分配更均匀”不自动等于更正义；权利、需求、责任和基线也可能不同。
- 它不能独自告诉决策者在所有冲突中应采用哪一种公平标准。

## 新场景测试

### 合理迁移

平台调整佣金使总体交易额上升，但成本集中到议价能力最弱的商户。拆分各群体的收益、损失和恢复能力，会改变对“增长成功”的判断。

### 错误迁移

两位同事选择不同午餐，并不存在共同政策造成的系统性成本转移；仅有结果不同不足以调用分配正义。

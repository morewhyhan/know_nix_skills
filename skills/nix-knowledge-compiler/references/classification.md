# CLC 全表检索与规范标签

本文件是编译器的 CLC 路由表和 `tags` 唯一词表。分类只负责检索与导航，不证明机制成立，也不替代专业术语核验。

## 强制检索协议

1. 每次 compile、reuse、query、validate 或 refactor，先扫描 A、B、C、D、E、F、G、H、I、J、K、N、O、P、Q、R、S、T、U、V、X、Z 共 22 个基本大类，不能从几个常用学科直接起步。
2. 对可能相关的大类展开到本表所列最细正式类号；按机制的调用问题分类，不按来源表面词汇分类。
3. `tags` 只能选择下列带反引号的 CLC 码与正式类名，使用 1–3 个最主要方向；不得写“组织行为学”“社会心理学”“博弈论”等无 CLC 码的自定义标签。
4. 跨学科术语仍可用于术语核验、别名和正文，但必须先映射回一个或多个 CLC 正式方向。
5. 本表没有列到所需深度时，使用最接近的上位正式类号，并在 `CLC方向` 说明待核实的下位类；不得自行发明 CLC 码。

## 边界说明

- 本表覆盖 CLC 五大部类、22 个基本大类，以及本项目随附第五版结构底本中列出的下位类目；不再用人工挑选的“常用方向”代替全表。
- CLC 完整版具有数万级深层类目。本地底本未列出的更深类号只能在核对权威 CLC 表后补充；在此之前使用本表中的最近上位类。
- 名称必须保持 CLC 含义。例如 `I04` 是“文学创作论”，不能写成“叙事学”；`Q98` 是“人类学”，不能写成“进化论与进化心理学”。

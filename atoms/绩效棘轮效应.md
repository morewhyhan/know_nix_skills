# 绩效棘轮效应

## 场景用途

当需要判断一次额外承接或初期超额表现为什么会转化为持续工作要求时调用本机制。它用于检查观察者是否把一次性、阶段性的高投入当成了稳定能力与长期意愿。

## CLC方向

CLC全表扫描：已检查 A、B、C、D、E、F、G、H、I、J、K、N、O、P、Q、R、S、T、U、V、X、Z 22 个基本大类。

主类：C93/管理学、F0/经济学、B84/心理学。

展开与排除：重点展开绩效目标调整、劳动激励与锚定判断。“锚定效应”只能解释早期样本影响判断；“绩效棘轮效应”还能表达历史高绩效抬高后续要求且向下难调，因此更贴近完整函数。

## 定义

在重复分配关系中，如果行动者的首次额外承接或早期超额表现没有被标明为临时例外，分配者可能把已展示水平上调为常态能力与意愿基线；当行动者回落时，态度化评价和恢复压力又促使其重返旧水平，使基线向上易调、向下难调。

## 边界

本机制不是“努力必然吃亏”，也不把所有标准提升都视为不当。它不同于正式岗位调整，因为新基线可能没有经过明确协商；不同于能力真实成长，因为判断对象是预期如何更新；也不同于沉没成本锁定，因为行动者持续承担不需要由舍不得既有投入来解释。

## 机制核心

函数签名：持续互动初期的额外承接或超额表现 -> 分配者上调常态基线并把回落解释为态度或能力下降 -> 恢复压力使旧基线不易下调并维持同类任务的重复分配

触发条件：分配者能够观察早期表现，双方将持续互动，且超额表现没有被明确标记为临时例外
  → 中介机制：已展示水平成为常态参照；回落被重新归因为态度或能力问题，并产生恢复到旧水平的评价压力；恢复行为又反向确认该基线
    → 可观察结果：同类任务持续集中，基线可以因新高表现上调，却难因回归可持续水平而下调
      → 适用边界：任务不会重复、临时边界已被共同确认，或后续基线经过显式重新协商时，棘轮不成立

## 参与角色

| 角色 | 在机制中的职责 |
|---|---|
| 表现者 | 在初期承担额外任务或给出高于可持续水平的表现 |
| 分配者或评价者 | 观察表现，并据此更新未来任务和评价基线 |
| 初始样本 | 被当作能力与意愿证据的一次承接或一段早期表现 |
| 后续基线 | 分配者据初始样本形成的常态预期 |
| 恢复压力 | 回落后由态度化或能力化评价产生，推动表现者恢复旧水平并使基线获得再次确认 |

## 成立条件

- 双方存在可重复的任务分配或绩效评价关系。
- 初始表现明显、可被分配者观察并归因于表现者。
- 临时支援、试用冲刺或一次性资源没有被明确标注。
- 分配者会用历史表现推断未来能力或意愿。
- 回落会影响评价或后续互动，使表现者承受恢复旧水平的压力；若回落没有任何反馈，只有锚定而不构成棘轮闭环。

## 迁移边界

**可迁移到：** 销售目标、团队支援、志愿服务或其他会根据历史表现重复分配任务的场景；来源之外的领域目前仅为机制推演。

**不能迁移到：** 单次交易、任务完全不可重复、超额投入已书面限定期限、后续标准由正式协商重新设定，或能力增长确实使新水平可持续的场景。

**假迁移警惕：** 后续工作量增加不一定来自基线棘轮，也可能来自正式晋升、总体需求上升或资源减少。只有当分配者以早期已展示水平作为常态参照，并据此解释后续回落时，才属于本机制。

## 复用价值

本 atom 将“第一次做了以后都是你的”和“试用期表现太好会拉高预期”合并成一条可检查的非对称更新链：高表现可抬高基线，回落却会触发态度化解释和恢复压力，使旧基线难以下调。

调用方式：分析持续加码时，依次检查初始样本是否超常、是否标注临时性、谁更新了基线、后续任务是否因该基线集中，以及回落是否被相对评价。

## 脱域说明

原领域是作者描述的额外工作和试用期表现。脱域时删除了“以后都会是你的”“只会拉高”等绝对措辞，也不沿用正式编制、体制内外等身份结论；保留的是“早期可见样本—基线上调—回落被态度化解释—恢复压力确认旧基线”的闭环。其他领域仅为角色映射。

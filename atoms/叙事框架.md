# 叙事框架

## 场景用途

当需要解释"为什么不同人面对同一情境会有截然不同甚至不切实际的预期和行为"时，用这个概念识别个体通过反复接触特定类型叙事（文学、影视、社交媒体等）而内化的、未经现实检验的默认认知模板。

## 定义

叙事框架是个体通过反复接触特定类型的故事而内化的一套关于"事情应该如何发生"的前意识认知模板。它规定了哪些行为是"正常的""浪漫的""理所当然的"，不是个体有意识持有的信念，而是被故事形式植入的默认期待。

与"价值观"的区别：价值观是可以被明确表述和辩护的规范性立场；叙事框架往往未被明确检视，个体意识不到自己持有它。与"刻板印象"的区别：刻板印象是对人群的简化归类；叙事框架是对事件因果链和角色行为的默认脚本。与"经验框架"的区别：经验框架来自个体亲历的直接经验；叙事框架来自外部叙事的间接植入。

排除：具体叙事内容（那是领域内容）、个人偏好（那是心理特质）、基于理性计算的预期（那是前瞻判断）。

在此场景中，叙事框架特指由文学作品和大众媒介反复强化而植入的、未经现实检验的关系预期模板——例如"存在一个命中注定的人""爱是无条件的""对方应该懂得我的全部需求"。

## 边界

- 不是"信念系统"：信念系统包含可被反思和辩护的命题，叙事框架可以前意识运作。
- 不是"世界观"：世界观覆盖本体论和认识论，覆盖面更广；叙事框架更窄，只涵盖特定类型情境的默认脚本。
- 不是"意识形态"：意识形态具有政治性和系统性；叙事框架可以是非政治的、碎片化的叙事残留。
- 排除直接经验形成的认知模式——那是经验框架。
- 排除有意识选择和持有的价值观立场——叙事框架的特点恰恰是持有者不知道自己在持有它。

## 机制核心

函数签名：反复出现且因果结构相似的外源叙事 -> 接收者压缩其中的角色关系与事件顺序为默认脚本 -> 相似现实情境自动调用该叙事框架

触发条件：个体反复接触结构相似的故事且缺少持续校准
  → 中介机制：重复的角色与因果模式被压缩、内化为前意识默认脚本
    → 可观察结果：个体在同类现实情境中按该脚本形成预期与判断
      → 适用边界：只有一次暴露、叙事结构不一致或直接经验持续修正脚本时，该链条不成立

```text
输入：反复出现、因果结构相似的外源叙事
  → 操作：个体把叙事中的角色关系与事件顺序压缩为默认脚本
    → 输出：面对同类现实情境时被自动调用的叙事框架
```

这里描述的是框架这一认知产物的形成接口；完整的反复暴露与内化过程由 [[叙事框架涵化]] 承载。

## 参与角色

| 角色 | 说明 |
|------|------|
| 叙事接收者 | 接触并可能内化外源叙事的个体 |
| 叙事媒介 | 提供重复角色、事件与因果脚本的文学、影视或其他媒介 |
| 默认脚本 | 被压缩并在相似情境中自动调用的认知模板 |

## 成立条件

- 个体反复接触结构相似的叙事，而非只接触一次孤立故事。
- 叙事提供了可被抽取的角色、行动与结果模式。
- 个体尚未用直接经验或批判性反思持续修正该模式。

## 迁移边界

**可迁移到：** 存在反复外源叙事、接收者和默认脚本的职业期待、成功叙事、家庭叙事或身体意象领域；这些领域当前均为机制推演。

**不能迁移到：** 由直接经验、明确论证或有意识价值选择形成的判断，因为其输入不是反复叙事形成的前意识脚本。

**假迁移警惕：** 仅把亲密关系中的具体期待替换为职业或家庭词汇，不等于机制迁移；必须重新确认反复暴露和默认脚本形成过程。

## 复用价值

作为"叙事框架涵化"机制的核心构件。当需要分析媒介叙事对受众认知的塑造作用时被调用。未来可被更大复合原子引用，如"社会期待生成系统""文化脚本与个体行为偏差"。是 L3 机制原子 [[叙事框架涵化]] 的组件。

调用方式：当我需要解释"某人为什么对某类关系/职业/生活方式有不切实际的期待"时，追溯其接触的叙事内容是否内化为了叙事框架。

原领域为亲密关系中的爱情叙事。提取中去掉了爱情领域的专属内容（恋爱、浪漫、婚姻等），保留了"外源叙事→反复暴露→内化→前意识认知模板"的结构。此概念可迁移到：职业期待领域（职场剧→职业叙事框架）、成功叙事领域（成功学→财富叙事框架）、家庭叙事领域（家庭剧→亲子关系叙事框架）、身体意象领域（社交媒体→外貌叙事框架）。

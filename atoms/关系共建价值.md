# 关系共建价值

## 场景用途

当需要判断"一段关系是否稳定""为什么有人可以随时离开而有人不能"时，用这个概念衡量双方共同投入所创造的价值总量——而不是只看某一方单方面的付出多寡。其核心功能是区分"感动自己的投入"与"双方共同建设的、不可分割的关系产出"。

## 定义

关系共建价值是关系双方通过双向的共同投入（时间、资源、情感、信息、社会资本等）而创造出来的、任何一方单独行动都无法产生的价值集合。它的两个关键特征是：**双向投入**（不是单方面行为）和**不可分割性**（价值附着于关系本身，关系结束即消失，不能由单方面带走）。

与"单方面投入"的区别：单方面投入是我放进去的东西，我离开后理论上可以收回（即使有沉没成本）；共建价值是"我们"共同创造的，关系结束它就消失。与"共同利益"的区别：共同利益可以是客观存在的、不需要双方投入就能享用的；共建价值需要双方持续的共同投入才产生。与"沉没成本"的区别：沉没成本是已经沉没的历史投入，不能收回；共建价值是当下的、持续的、需要双方维持的关系产出流。

排除：单方面的付出（无论多大）、外部赋予的身份（如夫妻名义、合伙人头衔）、仅仅是利益一致性（没有共同投入）、潜在的/未实现的合作价值。

## 边界

- 不是"共同财产"：共同财产是法律概念，可以分割；共建价值是非物质的关系产出流，不可分割。
- 不是"感情"或"爱"：感情是主观情绪体验，共建价值是客观的、双方行为可观测的关系产出。
- 不是"利益一致性"：利益一致是自然禀赋或外部条件，共建价值是人际互动的产物。
- 排除"单方面自我感动"的投入——如果对方没有参与、回应和互惠，你的付出不产生共建价值。

## 机制核心

函数签名：关系双方可互相响应的实际投入 -> 投入在互动中组合为单方无法等价产生的联合产出 -> 形成附着于关系本身的共建价值

触发条件：至少两方都投入资源或行动且能够接续彼此的投入
  → 中介机制：互相响应的投入发生组合，产生不可由单方独立复制的联合产出
    → 可观察结果：关系中出现双方共同维持、关系结束即受损的共享成果
      → 适用边界：只有单方面付出、投入互不关联或产出可由单方独立产生时，不形成共建价值

```text
输入：关系双方可互相响应的投入
  → 操作：投入在互动中组合成任何一方单独无法产生的联合产出
    → 输出：附着于关系本身的共建价值
```

该原子只承载共建价值的生成，不直接断言共建价值必然使关系稳定；稳定后果由 [[价值共建-关系稳定]] 承载。

## 参与角色

| 角色 | 说明 |
|------|------|
| 关系参与方 | 至少两个能够投入并回应彼此的行动者 |
| 双向投入 | 双方实际提供并被对方接续使用的资源或行动 |
| 联合产出 | 由互动生成、不能归于任何一方单独行为的结果 |

## 成立条件

- 至少两方都有可观察的实际投入。
- 一方投入能够被另一方回应并进入共同过程。
- 产出不能由任何一方单独等价地产生。
- 关系名义或单方面付出本身不算共建。

## 迁移边界

**可迁移到：** 存在至少两方实际投入、互动组合和不可由单方等价产生联合产出的商业合伙、团队、学术合作或社区共建；这些领域当前均为机制推演。

**不能迁移到：** 一次性交换、单方面赠与、只有共同利益但没有互动产出的场景，因为不存在双向组合生成的关系专属价值。

**假迁移警惕：** 共同出现、共同拥有或一方投入很多不等于共建价值；判定标准是双方投入是否组合成单方无法等价产生的联合产出。

## 复用价值

作为"价值共建-关系稳定"机制的依赖概念。当需要区分"什么真正绑定了关系"和"什么只是自我感动"时，调用此概念。未来可被更大复合原子引用，如"合作关系的稳定性条件""联盟存续的基础"。

调用方式：当我需要解释某段关系的稳定性时，检查双方的共建价值总量——而不是某一方的沉没成本或主观感情强度。

## 必要说明

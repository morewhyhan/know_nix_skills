# 专挑软柿子

## 场景用途

当必须完成的额外任务没有明确承接者，而分配者又难以用正式奖惩直接分配时，调用本机制判断询问与劝说是否会优先指向规则信息较少、边界识别较弱或更难拒绝的成员。

## CLC方向

CLC全表扫描：已检查 A、B、C、D、E、F、G、H、I、J、K、N、O、P、Q、R、S、T、U、V、X、Z 22 个基本大类。

主类：C93/管理学、C97/劳动科学、C91/社会学。

展开与排除：重点展开组织管理、劳动分配与社会互动方向；B84/心理学只能解释个体判断，不能单独承载任务对象选择结构。该机制回答成员承接阻力不均时，分配者如何选择更容易接受额外任务的目标。

## 定义

当额外任务必须完成、正式奖惩或强制能力受限且成员掌握规则与拒绝成本不同，分配者可能优先接触信息较少、较难拒绝的成员，使任务询问与劝说对象向这些低阻力成员集中；是否实际承接由后续促成机制决定。

## 边界

本机制不等于“所有新人都会被加派”，也不把新人身份本身当成原因；有效变量是可观察的信息、边界识别与拒绝阻力差异。它不同于随机分配、按专业能力匹配或正式轮值，也不推断分配者必然具有恶意。

## 机制核心

函数签名：必须完成但缺少正式分配抓手的额外任务与成员阻力差异 -> 分配者比较规则信息、边界识别和拒绝阻力并优先接触低阻力成员 -> 任务询问与劝说对象向较易承接者集中

触发条件：额外任务必须找到承接者，正式奖惩或强制能力受限，且分配者能在多个阻力不同的成员中选择
  → 中介机制：预期协商成本和被拒风险引导分配者优先选择规则信息较少、边界识别较弱或拒绝代价较高的成员
    → 可观察结果：额外任务的询问与劝说不成比例地指向低阻力成员；本机制本身不保证其接受
      → 适用边界：任务按透明规则轮值、承接资格完全由专业能力决定、成员阻力无差异或分配者不能选择对象时，该机制不成立

## 参与角色

| 角色 | 在机制中的职责 |
|---|---|
| 分配者 | 必须为额外任务寻找承接者，并估计不同成员的拒绝风险 |
| 候选成员 | 在规则信息、边界识别和拒绝成本上存在差异 |
| 低阻力成员 | 被分配者判断为更可能接受询问或劝说的候选者 |
| 额外任务 | 必须完成但缺少明确正式承接者或直接分配抓手的工作 |

## 成立条件

- 存在必须完成且承接归属不清的额外任务。
- 分配者缺少或认为自己缺少足以直接强制承接的正式抓手。
- 候选成员在规则知识、边界识别或拒绝成本上存在可被感知的差异。
- 分配者能够选择先接触谁，且低阻力成员至少有一定接受可能。

## 迁移边界

**可迁移到：** 团队临时支援、社区杂务或其他“任务必须完成但承接者可选择”的重复协作场景；来源之外目前均为机制推演。

**不能迁移到：** 透明轮值、随机抽签、依法或依合同明确归属、只按不可替代专业资格分配，或所有成员拥有相同信息与拒绝成本的场景。

**假迁移警惕：** 新人承担更多任务不自动证明发生了低阻力筛选；也可能是培训设计、岗位职责或能力匹配。必须观察分配者是否因较低拒绝风险而优先接触该成员。

## 复用价值

本 atom 将“找新人干活”的群体标签收窄为可检查的接触对象选择机制，把分析重点从身份偏见转到任务归属、正式分配抓手和成员拒绝阻力差异。

调用方式：分析额外任务集中时，依次检查任务是否必须完成、是否缺少正式承接规则、分配者能否选择对象、成员阻力如何不同，以及首次接触是否系统性偏向低阻力者。

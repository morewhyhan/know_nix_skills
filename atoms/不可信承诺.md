# 不可信承诺

## 场景用途

当需要判断一个未来回报暗示是否足以支持当前投入时调用本机制。它帮助区分“发话者能够影响结果”与“发话者能够兑现结果”，避免把建议权、推荐权或模糊好意当成可执行对价。

## CLC方向

CLC全表扫描：已检查 A、B、C、D、E、F、G、H、I、J、K、N、O、P、Q、R、S、T、U、V、X、Z 22 个基本大类。

主类：C93/管理学、F0/经济学、B84/心理学。

展开与排除：重点展开管理权限、承诺可信度与判断偏差；D9/法律只适用于承诺已形成法律义务的情形。该机制回答多级决策环境中，为什么接受者会高估非最终控制者给出的机会暗示。

## 定义

当接受者缺少结果分配规则的信息，而发话者只拥有建议权或有限影响力时，接受者可能把模糊的未来回报暗示误估为可兑现承诺，因而先行投入劳动或其他资源，但投入与最终回报之间没有可执行的绑定。

## 边界

本机制不把所有鼓励或发展建议都判为欺骗，也不推断发话者必然有操纵意图。它不同于正式强制，因为接受者不是在可执行制裁下行动；不同于有效绩效契约，因为回报条件和兑现主体没有被明确绑定；也不同于单纯结果不确定，因为关键问题是接受者误判了发话者对结果的控制程度。

## 机制核心

函数签名：信息不足的接受者收到非最终控制者给出的未来回报暗示 -> 把有限影响力误估为结果兑现权并高估承诺价值 -> 先行投入资源而实际回报与投入脱钩

触发条件：接受者重视某个未来结果，发话者能影响但不能单独决定该结果，且暗示没有明确兑现主体、条件和可核验节点
  → 中介机制：信息差使接受者混淆建议权、影响力与最终控制权，按高于实际可兑现性的价值评估暗示
    → 可观察结果：接受者为机会暗示承担额外任务，而结果仍由空缺、上级或其他程序决定，回报可能不发生
      → 适用边界：发话者拥有明确兑现权、承诺形成可执行交换，或接受者已按真实权限折价时，该链条不成立

## 参与角色

| 角色 | 在机制中的职责 |
|---|---|
| 接受者 | 评估未来回报并决定是否先行投入资源 |
| 发话者 | 给出机会或回报暗示，但不单独控制最终结果 |
| 最终控制者或约束 | 通过空缺、上级决策、集体程序或其他规则决定结果 |
| 模糊承诺 | 连接当前投入与未来回报，却缺少明确兑现主体、条件或检查节点 |

## 成立条件

- 接受者确实看重被暗示的未来回报。
- 发话者不拥有该结果的单方最终控制权。
- 接受者不能充分观察权限结构或结果分配规则。
- 暗示足以影响当前投入，却不足以形成可核验、可执行的交换。

## 迁移边界

**可迁移到：** 任何存在多级审批、代理人推荐和接受者信息不足的组织承诺场景；除来源中的职业场景外，其他场景目前都只是机制推演。

**不能迁移到：** 发话者拥有明确授权并给出可核验条件的正式承诺、没有未来回报暗示的直接命令，或接受者明知承诺只能作为低权重建议仍自主投入的场景。

**假迁移警惕：** 结果最后没有发生，不等于本机制成立。必须同时检查发话者是否缺少最终控制权、接受者是否高估了该权力，以及当前投入是否由该高估推动；单纯运气不佳或条件未达成不是控制权错配。

## 复用价值

本 atom 把“画大饼”的情绪标签改写为可检查的权限与承诺结构，使未来分析可以先核对谁控制结果、承诺绑定了什么，再判断当前投入是否有对价。

调用方式：遇到以未来机会换取当前投入的请求时，依次检查最终决策者、发话者权限、兑现条件、可核验节点，以及接受者是否把影响力误当成控制权。

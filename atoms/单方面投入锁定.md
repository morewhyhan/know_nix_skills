# 单方面投入锁定

## 场景用途

当需要解释"为什么那个付出更多的人反而是更害怕失去的一方""为什么有人已经明显被亏待却仍然不愿离开""为什么单方面的讨好和牺牲不仅不能绑定对方，反而让自己变得极其脆弱"时，使用此机制。它揭示了不对称投入如何通过沉没成本心理锁定投入方，而使其在关系中丧失议价能力和退出自由。

## CLC方向

F0 经济学（沉没成本效应 + 博弈论中的承诺-锁定机制。此机制锚定在"不对称投入→沉没成本心理锁定→退出成本不对称→议价能力丧失→被剥削风险"的因果链中。）

## 定义

单方面投入锁定是：在一段自愿关系中，当一方的投入（时间、资源、情感、机会成本等）持续远超另一方时，投入方因沉没成本心理（"我已经投入这么多，走了就全白费了"）而被锁定在关系中；而投入极少的一方因为几乎没有沉没成本，可以随时以极低代价退出。这种退出成本的不对称使投入方丧失议价能力，并为被剥削创造了条件——对方知道"你比我更离不开这段关系"。

与"忠诚"的区别：忠诚是基于价值和选择的有意识承诺；投入锁定是被沉没成本心理驱动的非理性滞留——表面看起来像忠诚，实则是被锁住了。与"投资"的区别：投资是期待回报的理性行为，有退出策略；投入锁定中的投入往往已经超出了理性投资的边界，变成挽救沉没成本的自我绑架。与"牺牲"的区别：牺牲可以是有意识的价值选择且可能增进共建价值；投入锁定中的投入不产生共建价值（因为对方没有参与建设），只是单方面的资源流失。

排除：双向共同投入（那是价值共建）、有意识的价值牺牲（那是选择性行为）、为获得对方回应的理性投资（那是投资策略）。

## 边界

- 不是"被PUA"或被精神控制：PUA/精神控制涉及有意的操纵策略和认知扭曲；投入锁定可以完全在没有恶意操纵的情况下发生——只是人性的沉没成本效应在起作用。
- 不是"爱得太深"：爱是情感体验，投入锁定是结构性的不对称——一个人可以不那么爱但被锁定（因为投入太多），也可以爱得很深但不被锁定（因为投入不过度且有退出选项）。
- 不是"共建价值"的高投入版本：共建价值的关键是双向投入+不可分割的产出；投入锁定是单向投入+无共建价值。见 [[价值共建-关系稳定]] 的 contrasting case。
- 排除双方投入虽不对等但差异不大且投入方随时可退出的情况——这是正常的关系不对称，不是锁定。

## 机制核心

```text
触发条件：关系中一方开始进行超过对方回应水平的持续单向投入
  → 投入累积：投入方的沉没成本不断增长
    → 心理锁定：沉没成本效应激活——"我已经投入这么多，现在退出之前的投入就全白费了"
      → 退出成本不对称：
          投入方：高退出成本（损失所有沉没投入 + 损失"未来可能回本"的希望）
          少投方：低退出成本（几乎没什么可损失的）
        → 议价权力转移：
            投入方："我更需要这段关系继续" → 丧失议价能力
            少投方："我随时可以走" → 获得议价优势
          → 两种恶化路径：
              ├── 路径A（被动脆弱）：少投方在外部诱惑/冲突发生时轻松离开
              │      → 投入方承受全部损失 → 被抛弃后的二次伤害（不只是失去关系，还有沉没投入的全部作废）
              │
              └── 路径B（主动剥削）：少投方意识到权力不对称
                    → 利用"你爱我就要……"的逻辑持续索取
                      → 投入方为了不让之前的投入白费，继续追加投入
                        → 恶性循环：越投入越被锁定 → 越被锁定越需要继续投入来"挽救"
                          → 终局：投入方被榨干后仍然被抛弃，或永久困在被剥削的关系中
```

关键洞察：单方面投入不产生 [[关系共建价值]]。共建价值=0时，即使投入方投入了全部身家，对方离开时也不损失任何不可分割的东西。

函数表达：
- 投入方退出成本 = f(沉没投入总量, 替代方案质量)
- 少投方退出成本 ≈ 0
- 当退出成本不对称度过临界值 → 关系进入锁定状态 → 任何有损投入方利益的互动都可能被接受

## 参与角色

| 角色 | 说明 |
|------|------|
| 投入方 | 持续进行远超对方水平的单向投入的个体。被沉没成本心理锁定 |
| 少投方/对方 | 投入极少或零投入的一方。拥有低退出成本和不对等议价优势 |
| 沉没投入 | 投入方已经投入且不可收回的资源（时间、金钱、情感、机会成本等） |
| 沉没成本心理 | 投入方因"投入已经太多"而不愿退出的心理机制。即使理性上应该退出，心理上无法接受损失 |
| 退出成本不对称 | 投入方退出代价 >> 少投方退出代价。这是权力和议价能力转移的结构性根源 |
| 关系 | 承载不对称投入的容器。因为缺乏共建价值，关系本身的纽带极其脆弱——仅由投入方的锁定维持 |

## 成立条件

- 关系是自愿的——双方都能选择退出（否则不存在"锁定vs自由"的对比）
- 投入方的投入远超对方的回应/投入水平，且持续了一段时间
- 投入方受到沉没成本心理的影响（大多数人类会，但程度因人而异）
- 少投方有真实的退出选项（不是理论上能离开但实际不能）
- 投入方的投入并未产生共建价值——即对方没有参与进来共同建设

如果双方投入对等、或投入方没有沉没成本心理、或投入产生了共建价值（转为价值共建-关系稳定机制的运作域），则此机制不触发或转换为其他机制。

## 迁移边界

**可迁移到：**
- 企业-供应商关系：供应商为企业定制生产线/专有技术（沉没投入）→企业可以随时更换供应商但供应商几乎不能换客户→供应商被迫接受日益苛刻的条款
- 国际关系（小国-大国）：小国在安全/经济上全面依赖某大国（单方面投入）→大国可以利用小国的依赖提取让步→"芬兰化"（Finlandization）是温和版本，军事/经济附庸是极端版本
- 就业关系：员工为公司投入了高度专用化的技能（只在当前公司有价值）→公司可以用低薪/差条件留住员工→员工不敢离职因为技能在其他地方不值钱
- 友情中的单方面维持者：一方持续组织聚会/主动联系/提供帮助→另一方从不主动但被维持→维持者怕失去这段友谊而不敢停止投入
- 粉丝-平台/偶像：用户在某平台积累了大量的内容、关注者、数据→平台提高抽成/改变规则→用户因沉没成本难以迁移

**不能迁移到：**
- 双方投入对等的关系——不存在退出成本不对称
- 投入方完全理性、不受沉没成本效应影响的情况（理论上存在但现实中罕见）
- 投入产生了共建价值的关系——转入了价值共建-关系稳定的运作域
- 强制性的、不能退出的关系——不涉及"锁定vs自由"的选择
- 一次性交易——没有持续的投入积累过程

**假迁移警惕：** 不要把"我为你做了很多"和"我被你锁定了"混为一谈。一个人为另一个人做了很多事但没有失去退出选项（比如随时可以停止、有替代选择、投入没有沉没）时，不是投入锁定。锁定的必要条件是：沉没成本足够大 + 替代方案不可接受 + 对方可以利用这种不对称。

## 复用价值

这是原文"不要单方面傻瓜式投入""你投入全部身家但是对方不付出，那你们的共建价值就是零"的机制化表述。它将这个直觉提炼为可操作的诊断工具：判断一段关系是否进入了投入锁定状态。可应用于亲密关系、商业关系、国际关系和任何自愿合作关系。

调用方式：当我需要判断"这个人为什么离不开一段明显糟糕的关系"时，检查是否存在退出成本不对称——具体来说，看他/她投入了多少对方没有参与建设的东西。

## 领域绑定

| 领域 | 投入方 | 少投方 | 沉没投入 | 锁定表现 | 剥削形式 |
|------|--------|--------|---------|---------|---------|
| 亲密关系 | 过度付出的一方 | 接受但不回应的一方 | 时间/金钱/情感/机会成本 | 明知被亏待也不离开 | 持续索取（彩礼/消费/情感服务） |
| 供应商-企业 | 专用化供应商 | 大客户企业 | 定制产线/专有技术 | 被迫接受降价/苛刻条款 | 逐步压价、拖延付款 |
| 小国-大国 | 依附小国 | 主导大国 | 安全/经济全面偏向 | 在大国压力下让步 | 提取政治/经济让步 |
| 员工-公司 | 专用化员工 | 公司 | 公司专用技能/人脉 | 忍受差待遇不敢跳槽 | 低薪/超时工作/零晋升 |

当前主要在亲密关系领域中验证。企业-供应商和国际关系为部分验证。

## 脱域说明

原领域为亲密关系中"单方面傻瓜式投入"。提取中去掉了：爱情/伴侣/彩礼/买房写名等专属内容，保留了：不对称投入→沉没成本→退出成本不对称→议价能力丧失→脆弱/被剥削弱。该机制不依赖任何亲密关系专属概念，可迁移到任何"自愿但有退出选项、涉及持续投入的非强制性关系"。

## 区分说明

atoms/ 中暂无旧原子可复用。

本批原子"价值共建-关系稳定"与本原子构成对称对照（[[价值共建-关系稳定]] - contrasts_with）：正面路径→双向共建→稳定；反面路径→单向投入→锁定→脆弱。两者的关键区分变量是投入是否为双向——双向投入产生共建价值（稳定），单向投入产生锁定（脆弱）。

本批原子"托付心态"与本原子亦构成对称对照（[[托付心态]] - contrasts_with）：托付心态是"你应该为我付出"（向对方索取），投入锁定是"我为你付出一切"（自己过度给予）——两者方向相反但都导致关系失衡，且常常同时出现在同一段关系中（一方托付索取、另一方投入锁定）。

本批原子"激情衰减-结构维系"为本原子提供了适用范围限制（[[激情衰减-结构维系]] - constrains）：投入锁定机制在激情驱动期可能被爱情激素掩盖——投入方在荷尔蒙阶段不觉得自己被锁定，只有在激情衰减后才发现自己已经出不去了。

## 必要说明

学科来源：沉没成本效应（sunk cost effect）是行为经济学中 Kahneman & Tversky 前景理论（prospect theory）的经典发现——人们倾向于继续投入已亏损的项目以避免实现损失。投入锁定将沉没成本效应从一次性决策推广到持续性关系中的累积锁定过程。博弈论中的承诺（commitment）概念与此有交叉——投入锁定是一种非策略性的、心理层面上的被动承诺，而非主动的策略性承诺（如"破釜沉舟"）。

"共建价值=0时对方随时可以离开你"这个原文表述被本机制提炼为退出成本不对称的核心量化条件。

常见别名：沉没成本陷阱、关系锁定、自我绑架式付出。

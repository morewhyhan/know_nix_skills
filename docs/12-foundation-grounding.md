# 前置概念补齐：学科分类框架 v0.5

## 核心结论

机制原子要能跨领域复用，它依赖的概念本身也得是可复用的。如果一个上层机制引用了只在原领域里有含义的概念，它就不可能脱域。

所以要补齐前置概念。补齐的目标是：**上层机制引用的每个概念，都有独立于任何单一领域的清晰定义。**

## 这套东西是什么——不是什么

**是：** 一个分类框架。当从文章、对话中提取出某个概念或机制后，用它来判断"这个东西属于哪个学科方向"、"它的前置依赖应该从哪里找"。每个方向下列出的概念是**示例**——用来标示这个方向的典型内容、帮你感受它的边界。它们远远不是全部。

**不是：** 一个概念词典。不要以为只有表里写的那些词才属于这个方向。实际上任何学科下面都有大量概念——表里的例子只是"你在这里可能会遇到什么"的快照。

用法：遇到一个不熟悉的概念 → 判断它属于哪个 CLC 方向 → 去对应方向检索是否已有原子承载 → 没有就补齐。边界说明帮你判断"它到底属于 E0 还是 D0"、"它属于 Q14 还是 Q4"。

## 为什么从学科里找

学科里那些被反复验证过的中层概念（机会成本、反馈回路、贝叶斯更新、瓶颈、代价信号……），本身就经过了跨领域抽象。它们是从无数具体案例中提炼出来的——这恰好就是"脱域"。学科概念的可复用性不是来自学术血统，而是来自它们已经被证明了可以在截然不同的系统中成立。

但不要无限追溯。停止标准：当前概念已经是一个**完整的、能被独立使用的脱域概念**——定义清晰、边界清楚。再往下拆，拆出来的东西不能独立使用，只能作为这个词的释义零件。

## 分类框架

排列遵循 CLC（中国图书馆分类法）字母序列。**括号中的概念是示例，不是全部**——它们标记这个方向的典型内容，用来帮你感受边界。

| CLC | 方向 | 边界与范围 | 示例概念（非穷尽） |
| --- | --- | --- | --- |
| B0 | 哲学与认识论 | 关于"知道什么"和"存在什么"的底层框架。涉及因果结构、知识边界、实在与表象。不包括伦理判断（B82），不包括推理形式（B81），不包括科学方法论（N0）。 | 因果关系类型（INUS/反事实/概率因果）、奥卡姆剃刀、本体论范畴、归纳问题、身心问题、目的论与机械论 |
| B81 | 逻辑学 | 推理的有效形式。论证结构、条件关系（充分/必要/充要）、谬误识别、模态推理。不包括内容层面的真假判断（B0），不包括修辞说服（H0）。 | 充分条件与必要条件、演绎与归纳、谬误类型、悖论、模态逻辑、可能世界、非形式逻辑 |
| B82 | 伦理学 | 关于"应该做什么"的框架。规范判断、价值排序、道德推理。不包括事实判断（B0），不包括法律规范（D9）。 | 功利主义、义务论、德性伦理、公平原则、后果论、道德权衡 |
| B9 | 宗教学 | 信仰系统的结构与功能。仪式、神圣性、社群凝聚、教义传播。不包括科学解释（N0），不包括一般社会规范（C91）。 | 仪式、神圣与世俗、禁忌、皈依、卡里斯玛、教派与分裂 |
| C3 | 研究方法论 | 怎样生产可靠的知识。研究设计、测量、推断逻辑、有效性检验。适用于社会科学和行为科学。不包括具体统计技术（O21）。 | 量化与质性、操作化、信度与效度、实验与准实验、抽样逻辑 |
| C91 | 社会学理论 | 社会结构与社会行动的一般理论。宏观与微观、结构与能动性、建构与实在。不包括具体社会制度分析（分属D/K/F等）。 | 理想类型、社会建构、自反性、结构-能动性、方法论个人主义 |
| D0 | 政治学 | 权力的组织与正当化。统治、秩序、集体决策、国家与社会关系。不包括法律技术（D9），不包括军事手段（E）。 | 权力与权威、合法性、集体行动、公共品与搭便车、制衡与分权 |
| D9 | 法律与制度设计 | 规则的制定、解释与执行。程序正义、证据标准、权利义务、惩罚逻辑。不包括道德判断（B82），不包括政治权力分配（D0）。 | 先例、契约、举证责任、正当程序、管辖权、规则与裁量 |
| E0 | 战略理论 | 冲突中如何确定目标和总体路径。威慑、博弈、信息与不确定性。不包括具体战术执行（E8），不包括后勤技术（E9）。 | 威慑谱系、重心、战争迷雾、OODA循环、先发制人 |
| E8 | 战役与战术 | 冲突中如何配置和运用力量。集中分散、攻防转换、主动权。不包括战略层面（E0），不包括武器技术（E9）。 | 集中与分散、纵深防御、消耗战与机动战、包围与突破 |
| E9 | 军事技术与后勤 | 冲突的物质与技术基础。供给链、情报周期、技术突袭与应对。不包括战略（E0）或战术（E8）。 | 后勤链、情报周期、C4ISR、技术突袭与应对 |
| F0 | 经济学 | 稀缺条件下的选择与交换。激励、成本、效率、市场与非市场机制。不包括金融工具定价（F8），不包括管理执行（管理学）。 | 机会成本、边际分析、激励、交易成本、外部性、套利、帕累托改进 |
| F8 | 金融 | 跨时间的资源配置与风险管理。货币、信用、资产定价、风险转移。不包括实体经济运行（F0），不包括会计准则。 | 贴现、对冲、流动性、道德风险、期限错配、安全边际 |
| G0 | 文化理论 | 文化如何塑造行为和被行为塑造。符号、惯习、场域、权力-知识。不包括传播过程（G2），不包括教育过程（G4）。 | 文化资本、惯习、场域、亚文化、濡化、文化霸权 |
| G2 | 传播学与媒体 | 信息如何通过媒介到达受众并产生效果。议程、框架、把关、网络效应。不包括语言结构本身（H0）。 | 议程设置、框架理论、把关人、回声室、两级传播 |
| G4 | 教育学 | 知识、技能和态度如何被传递和获得。学习设计、评估、动机、认知发展。不包括认知机制本身（心理学）。 | 脚手架、最近发展区、间隔重复、形成性评估、迁移学习 |
| G8 | 体育与竞技 | 身体能力的训练与竞技表现。周期化、恢复、技能迁移。不包括生理机制（Q4），不包括团队管理（组织行为学）。 | 训练周期化、超量恢复、技能迁移与干扰、默契成本 |
| H0 | 语言学与语用学 | 语言的结构与使用。语义、语用、话语行为、隐喻。不包括叙事结构（I04），不包括传播效果（G2）。 | 语义场、语用含义、言语行为、预设、隐喻映射、语言相对论 |
| I0 | 文学理论 | 文学文本的性质、功能与阐释。体裁、接受、互文、陌生化。不包括叙事结构分析（I04）。 | 原型、陌生化、卡塔西斯、反讽、体裁规约、互文性 |
| I04 | 叙事学 | 故事的结构与叙述逻辑。视点、时间、可靠性。与I0的区别：I0关心"文学是什么"，I04关心"故事怎么讲"。 | 叙事弧、不可靠叙述者、框架叙事、聚焦与视角、时间倒错 |
| J0 | 艺术理论与美学 | 审美经验与艺术判断。美的性质、表现、崇高、趣味。不包括形式设计规则（J06）。 | 模式与变化、对比与协调、和谐与张力、崇高与优美 |
| J06 | 形式与设计 | 视觉与空间的构成规律。适用于平面、建筑、产品等一切形式的领域。不包括审美哲学（J0）。格式塔原理起源于知觉心理学（B84），但已成为设计领域的基础语言。 | 比例、留白、格式塔（知觉心理学借用）、信息密度、视觉层级、网格系统 |
| K0 | 历史学 | 关于过去的解释与方法。史料、时间、因果、记忆。不研究当下社会结构（C91），不涉及地理空间（K9）。 | 偶然性与必然性、反事实、档案偏见、辉格史观、长时段 |
| K9 | 地理学与空间分析 | 空间关系及其效应。距离、尺度、地方、扩散。不包括时间维度（K0），不包括生态系统（Q14）。 | 邻近性、中心性、边界效应、距离衰减、空间扩散、地方感 |
| N0 | 科学哲学与方法论 | 科学知识如何被生产和验证。范式、证伪、模型、实验、还原论。适用于自然科学。不包括社会科学的特殊性（C3），不包括一般认识论（B0）。 | 可证伪性、范式与科学革命、研究纲领、模型与实在、还原论与涌现论、科学划界 |
| N94 | 系统科学 | 跨学科的系统一般理论。边界、层次、涌现、整体论。适用于任何可以被视为系统的对象。 | 系统边界、层次与涌现、开放与封闭系统、整体论 |
| O1 | 数学、关系与图 | 抽象结构与形式关系。不依赖经验内容的纯粹关系。不包括统计推断（O21），不包括物理建模（O4）。 | 函数、集合、映射、层级、组合、网络节点、对称、不动点 |
| O21 | 概率与统计 | 不确定性的量化与推断。概率、分布、估计、检验。与O1的区别：O1是确定性结构，O21是随机性与推断。 | 基率、条件概率、贝叶斯更新、样本偏差、噪声、相关≠因果 |
| O23 | 控制论与信息论 | 信息、通信、控制的一般数学理论。香农熵、信道、编码、反馈控制。与O21的区别：O21研究统计推断，O23研究信息本身的结构和传输。不包括信息的经济学应用（信号博弈见博弈论），不包括信息的生物学应用（代价信号见Q98）。 | 信息熵、信号噪声分离、信道容量、编码与冗余、反馈控制 |
| O3 | 力学 | 力、运动与变形的规律。应力、疲劳、流体行为、稳定性。与O4的区别：O3研究宏观连续体的运动和变形，O4研究更基础的物理规律。 | 应力集中、疲劳断裂、流变、湍流、共振、失稳 |

| O6 | 化学 | 物质的组成、结构与转化。反应、催化、平衡、选择性。与O4的区别：O6研究对象在反应中改变组成。 | 催化、饱和、反应速率、链式反应、活化能、缓冲、化学平衡 |
| P1 | 天文学 | 天体与宇宙的结构和演化。轨道、引力、深时、宇宙尺度。 | 深时、轨道与周期、引力与结构 |
| P3 | 地球物理学 | 地球内部结构与动力过程。板块、地震、地磁。与P4的区别：P3是固体地球，P4是大气与气候。 | 均变论与灾变论、地层叠置、板块构造 |
| P4 | 大气科学与气候 | 大气过程与气候系统。环流、温室效应、临界点。与P3的区别：P4是大气圈，P3是岩石圈。 | 气候临界点、温室效应、大气环流 |

| Q3 | 遗传学 | 生物性状的继承与变异机制。基因、表达、选择、漂变。与Q98的区别：Q3研究遗传的分子和统计机制，Q98研究进化尺度的后果。 | 显性与隐性、表观遗传、基因漂变、重组、突变负荷 |

### 跨学科方向

以下方向横跨多个 CLC 大类，无法归入单一字母。排列不按 CLC 序列，按主题聚类。

| 方向 | 边界与范围 | 示例概念（非穷尽） |
| --- | --- | --- |
| 博弈论与机制设计 | 策略互动的形式分析。玩家、信息、策略、均衡、激励兼容。跨越经济学（F0）、数学（O1）、政治学（D0）、生物学（Q98）。 | 玩家、策略空间、收益函数、纳什均衡、承诺、信号博弈、机制设计 |
| 决策理论 | 个体如何在不确定下做出选择。规范性（应该怎么选）和描述性（实际怎么选）。跨越经济学（F0）、心理学（B84）、统计学（O21）。 | 期望效用、风险偏好、损失厌恶、时间贴现、前景理论 |

## 如何使用这个框架

**当遇到一个概念/机制时：**
1. 判断它属于哪个 CLC 方向。看"边界与范围"列——它回答什么核心问题？它和相邻方向的分界线在哪？
2. 去对应方向检索已有原子库，看是否已有承载。
3. 没有就按该方向的要求补齐——概念原子只需要定义+边界+一句话机制，机制原子需要完整四要素。
4. 不继续下钻。除非另一个上层原子确实需要。

**示例概念不是全部。** 表里列的词只是用来标示方向范围的标志物。任何一个方向下面都有几百个未列出的概念——但它们都属于同一个"知识区域"，共享同一套前置假设和分析工具。

**跨学科方向**标记为无CLC码。当遇到的概念同时属于多个方向时（比如"框架效应"同时属于心理学、决策理论、传播学），优先归入最相关的跨学科行，或者同时在多个方向检索。

---

## 补齐流程

1. 提取上层机制原子，列出它引用的所有概念和参与角色。
2. 对每个概念，在已有原子中检索是否有可独立理解的定义。
3. 有 → 写入 `depends_on`，链接过去。
4. 没有 → 用本框架判断它属于哪个方向 → 在对应方向下创建概念原子 → 链接。

## 概念原子的写法

比机制原子简单：

```markdown
# 概念名称

## 定义
一句话说清它是什么。

## 边界
它不是什么，容易和什么混淆。

## 一句话机制
它背后的因果结构（如果有）。

## 必要说明
常见别名、为什么由当前材料触发补齐。
```

不需要参与角色、成立条件、迁移边界——那些是机制原子的要求。概念原子的角色是被引用。

---

## 停止规则

- 补齐到概念能被独立理解即可。
- 不要追溯概念到更底层的概念（除非不追溯上层原子就无法理解）。
- 不要预先建一整套"学科知识库"——只补齐当前材料触发的。
- 完整 CLC 类目结构见 `docs/clc-reference.md`。

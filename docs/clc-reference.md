# 中国图书馆分类法（CLC）完整类目结构

> 来源：《中国图书馆分类法》第五版。简称"中图法"，英文缩写 CLC。
> 采用汉语拼音字母与阿拉伯数字相结合的混合号码，一个字母代表一个大类。

## 五大部类

| 部类 | 包含大类 |
|------|---------|
| 马克思主义、列宁主义、毛泽东思想、邓小平理论 | A |
| 哲学、宗教 | B |
| 社会科学 | C, D, E, F, G, H, I, J, K |
| 自然科学 | N, O, P, Q, R, S, T, U, V, X |
| 综合性图书 | Z |

## 与本项目的关系

这份参考文档是 `docs/12-foundation-grounding.md`（前置概念补齐）的 CLC 底本。`docs/12` 中的检索地图从这 22 个大类中筛选了**能产出脱域可复用概念的方向**——并非每个类目下的知识都能脱离原语境使用。

完整类目结构保留于此，作为：
1. 检索未知领域脱域概念时的全量参照
2. `docs/12` 检索地图的扩展基线
3. 判断一个概念"是否有学科归属"时的导航
